
# Primehub Instancetypes

primehub instancetypes `<verb>` `[args]` `[flags]`


## Available Commands

* Get
* List



#### Get

Get an instance type by name


```
primehub instancetypes get <name>
```
**Required Arguments**
* name
 


 



#### List

List instance types


```
primehub instancetypes list
```
